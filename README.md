# Smart Healthcare Expense & Disease Prediction System (Machine Learning Project)

This project is a **Machine Learning-based healthcare prediction system** that analyzes patient data to:
1) Predict disease risk using **classification models**
2) Estimate healthcare expenses using **regression models**
3) Provide **lifestyle improvement suggestions** based on data insights

The system is purely data-driven and operates on structured patient datasets. No hardware or IoT components are involved.
